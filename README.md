## 🚀 Proyectos Destacados

### 🌦️ Estación Meteorológica Inteligente
> IoT + Plataforma Web en Tiempo Real

Sistema completo que integra **sensores físicos con una aplicación web** para monitoreo climático en tiempo real.

- ⚙️ **Mi rol:** Desarrollo del firmware Arduino (C++) y construcción del backend que conecta el hardware con la web
- 🔌 **Stack:** Node.js · Express · Socket.io · PostgreSQL · Chart.js · Arduino
- 📡 Datos de temperatura, humedad, presión y viento transmitidos en tiempo real vía WebSocket
- 🖨️ Carcasa impresa en 3D para los componentes electrónicos

---

### 💊 SIPRODEM — Sistema de Pronóstico de Demanda de Medicamentos
> ExpoCiencias Nacional 2025 Tamaulipas · MILSET AMLAT 🏆

Plataforma de Machine Learning para **predecir el desabasto de medicamentos en hospitales públicos mexicanos** con 4 semanas de anticipación.

- 🤖 **Modelos:** Prophet (R²=0.94) + Random Forest (R²=0.99)
- 📊 Dashboard interactivo con alertas automáticas de desabasto en tiempo real
- 🗂️ ETL completo sobre datos históricos gubernamentales (2020–2024)
- 🏥 Orientado a IMSS, IMSS-BIENESTAR e INER
- 🌍 Contribuye a los ODS: Salud (3), Innovación (9) y Consumo Responsable (12)
